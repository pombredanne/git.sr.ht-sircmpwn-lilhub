Lil' Hub

This is a lightweight, free software frontend for GitHub.

  • Lightweight & JavaScript free
  • URL-compatible with GitHub
  • Free software

Simply replace github.com with lilhub.org in any GitHub URL to make use of Lil' Hub.

Want to see how it works? Check out this example repository: riscv/riscv-isa-manual
